5 HTML Tags I have learned

About me
<h1> tag
A text semantic tag used for the most important heading of a page.
<p> tag
Another text semantic tag used to create a paragraph.
<a> tag
Anchor tags are used to hyperlink to external websites, internal pages, documents or any other resource on the web or on our web server.
<img> tag
This HTML element displays an image on the webpage. This tag is different than other tags, in that it has no closing tag.
<form> tag
The form tag is used to create a form for user input.
